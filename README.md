# Metal Gear Peon

A [CESP 1.0](https://openpeon.com/spec) sound pack for [OpenPeon](https://openpeon.com) featuring sound effects from Metal Gear Solid.

## Install

```
peon packs install sambencivengo/metal-gear-peon
peon packs use metal-gear-peon
```

## Categories

| Category | Sounds |
|---|---|
| `session.start` | Codec Open |
| `task.acknowledge` | Item Open, Codec Close |
| `task.complete` | Item Used |
| `task.error` | Alert (!), Game Over |
| `input.required` | Codec Call |
| `resource.limit` | Door Buzz |
| `session.end` | Exit |
